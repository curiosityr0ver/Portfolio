# Project Screenshots Directory

This directory contains screenshots and thumbnails for portfolio projects.

## File Naming Convention

For each project, use the following naming pattern:

### Thumbnails
- `[project-name]-thumbnail.png` - Small preview image (recommended: 400x300px or similar)
  - Example: `vaultfolio-thumbnail.png`

### Screenshots
- `[project-name]-screenshot-1.png` - Main screenshot
- `[project-name]-screenshot-2.png` - Additional screenshot
- `[project-name]-screenshot-3.png` - Additional screenshot (if needed)
  - Example: `vaultfolio-screenshot-1.png`, `vaultfolio-screenshot-2.png`

## Supported Formats

- PNG (recommended)
- JPG/JPEG
- WebP (for better optimization)

## Image Guidelines

### Thumbnail Specifications
- **Recommended size**: 400x300px or 800x600px
- **Aspect ratio**: 4:3 or 16:9
- **File size**: Keep under 200KB for fast loading

### Screenshot Specifications
- **Recommended size**: 1920x1080px or 1280x720px
- **Aspect ratio**: 16:9 (landscape) works best
- **File size**: Keep under 500KB per image
